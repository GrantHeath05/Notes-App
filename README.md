# NotesApp

A simple note‑taking application built as a personal Java refresher and an introduction to **JavaFX** and **Maven**.  
This project focuses on core JavaFX concepts such as FXML layouts, controllers, event handling, and basic application structure.

## Features
- Write and save notes  
- Basic JavaFX UI using FXML  
- Maven‑managed dependencies  
- Lightweight foundation for future expansion (titles, sorting, JSON storage, note listing, etc.)

## Purpose
This project serves as:
- A hands‑on refresher of modern Java (Java 17+)  
- A first step into building desktop applications with JavaFX  
- A simple sandbox for experimenting with UI, controllers, and data persistence  

## Tech Stack
- **Java 17**  
- **JavaFX 21**  
- **Maven**  
- **Gson** (planned for JSON note storage)

## Running the App
Run the JavaFX application using Maven:
mvn javafx:run
